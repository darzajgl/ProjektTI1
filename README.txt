Polski

Prosta aplikacja webowa stworzona w ramach nauki języka PHP i obsługi bazy danych MySQL, jako projekt w ramach przedmiotu "Techniki Internetu". W projekcie zostały wykorzystane technologie: PHP, HTML, SQL i CSS.

Składa się ona z następujących części:
-strony menu
-strony logowania
-strony rejestracji z użyciem captcha do zabezpieczenia przed botami
-strony konta użytkownika

Użytkownik może:
-zalogować się, jeśli już posiada konto
-zarejestrować się, jeśli chce utworzyć nowe konto
-po zalogowaniu podejrzeć swoje dane

Po wcześniejszym zalogowaniu użytkownik jest automatycznie przenoszony ze strony logowania na stronę konta. Rejestracja nowego użytkownika powoduje wylogowanie poprzedniego użytkownika.

Aplikacja wykorzystuje sesję do przechowywania wprowadzonych danych. Połączenie z bazą danych MySQL umożliwia pobieranie i zapisywanie danych.

English

A simple web application created as part of learning PHP language and MySQL database management, as an academic project for the "Internet Techniques" subject. Technologies used in the project include: PHP, HTML, SQL and CSS.

It consists of the following parts:
-menu page
-login page
-registration page with captcha
-user account page

Users can:
-log in if they already have an account
-register if they want to create a new account
-view their data after logging in

After logging in, the user is automatically transferred from the login page to the account page. Registration of a new user logs out the previous user.

The application uses a session to store entered data. The connection to the MySQL database allows for data retrieval and saving.
